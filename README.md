# Running Locally

```bash
npm install vercel
npm i -g vercel
vercel dev
```
***or (recommended)***

```python
python -m flask --app api/app run -p 3000
```

Your application is now available at `http://localhost:3000`.

# Deploy to Vercel
After your commit, Check [API url](https://it-project2024-sm-2-git-api-may-bee.vercel.app/)

> [!IMPORTANT]
> Please ensure that you are not pushing any api keys to the repository.
